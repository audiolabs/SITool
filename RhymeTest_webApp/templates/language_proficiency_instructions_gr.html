Anweisungen für den Test der Sprachkenntnisse

Willkommen beim Test zur Sprachbeherrschung. In diesem Test werden Sie sich einen Audioclip anhören und dann das Bild auswählen, das dem Gehörten am besten entspricht.

Anweisungen:

  • Klicken Sie auf die Schaltfläche "Test starten" unten, um den Test zu beginnen.
  • Jede Frage besteht aus einem Audioclip, gefolgt von 3 Bildern.
  • Hören Sie sich den Ton genau an und klicken Sie auf das Bild, das dem Gehörten entspricht.
  • Sie können nur eine begrenzte Anzahl von falschen Antworten geben. Achte darauf, dass du richtig antwortest, um fortzufahren.

Sobald Sie bereit sind, klicken Sie auf die Schaltfläche "Test starten", um zu beginnen.
